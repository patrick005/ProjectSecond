## 프로젝트 개요

### 목표
- 클라이언트에서 서버로 접속 → 접속된 클라이언트 2개시 게임시작
- ESP8266에서 자이로센서 측정값을 통한 서버로 데이터 전송 → 공격, 회피
- 게임 로직을 통한 게임 결과 → LiteSQL 저장
- 게임 진행에 대해 GUI로 시각화

### 주요 기능
- 서버-클라이언트 1:2 통신
- 자이로 센서를 통한 움직임 방향성 제어
- 서버-클라이언트 실시간 통신
- ZMQ를 통한 각 모듈과의 송수신
